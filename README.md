# Syllabus

This unit relies heavily on C# and Windows .NET MVC web services to develop distributed networking applications and topics covered include
* Remote procedure calls and asynchronous calls
* Multi-tiered architectures for distributed computing
* Serialization and sending data over a network
* Development and use of web services and APIs
* Security in distributed systems
* Serverside Web scripting with CGI based approaches

# Assignment 1 -  Online Gaming Lobby Management System

Built an online gaming lobby management system for Mortal Combat X using **.NET Windows Communication Foundation (WCF)**

## Features:
- [ ] User management with unique username logins
- [ ] Lobby room management
- [ ] Private messaging and lobby room chats
- [ ] File sharing

# Assignment 2 -  Bank Data Web Application

Built a secure banking web application using ASP.NET Core

## Features:
- [ ] Business (web services) & presentation layers (Web UI)
- [ ] User & admin dashboards
- [ ] Users: profile management, transaction history & money transfer initiation
- [ ] Admin: profile management, user management, security controls, transaction logs
- [ ] REST API integration using RestSharp
